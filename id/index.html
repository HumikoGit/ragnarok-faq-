Ragnarok: Sharing Hero — FAQ

Pertanyaan yang sering diajukan tentang game, blockchain, dan hadiah.

🎮 Informasi Dasar Game
Apa itu game ini?
Ragnarok: Sharing Hero adalah game RPG blockchain berbasis IP Ragnarok. Game perang faksi berskala besar di mana faksi Cosmos dan Chaos bersaing menguasai Midgard secara real-time. Dikembangkan oleh Funigloo dengan pengalaman 17 tahun.
Kapan game ini dirilis?
Dirilis resmi pada 26 Maret 2026.
Di mana bisa mengunduh?
Tersedia gratis di Google Play Store dan Apple App Store. Cari "Ragnarok Sharing Hero".
Apakah gratis atau berbayar?
Unduhan dasar dan bermain gratis. Pembelian item dalam game dan perdagangan NFT mungkin memerlukan biaya tambahan.
⚔️ Cara Bermain
Bagaimana cara bermainnya?
Besarkan Knight Anda, perkuat perlengkapan, buru bos, dan bertarung melawan pemain lain secara real-time di medan perang Middle Land.
Apa itu PvP?
PvP adalah konten di mana pemain bertarung langsung. Faksi yang menguasai Middle Land memberikan buff reward pengumpulan 2x kepada seluruh anggota faksi.
Apa itu Cosmos dan Chaos?
Dua faksi dalam game. Jika satu faksi terlalu kuat, sistem secara otomatis mengaktifkan pembatasan pembuatan karakter untuk memulihkan keseimbangan.
Apa itu sistem Sharing?
Pinjam karakter teman dan masukkan ke Knight Order Anda untuk bermain konten bersama. Teman yang meminjamkan karakternya mendapat hadiah berbagi.
🔗 Blockchain / NFT
Apa itu Immutable?
Platform game blockchain khusus. Perusahaan global berbasis Australia didirikan 2018, memungkinkan pemain benar-benar memiliki item dalam game sebagai aset digital (NFT).
— Immutable Network: Jaringan blockchain untuk aset game
— Immutable Passport: Login game dan integrasi dompet (6 juta+ pendaftar)
— Immutable Play: Platform quest, hadiah, dan papan peringkat
Di mana membeli item NFT?
Di marketplace TokenTrove.
Bagaimana mendapatkan $MINT token?
$MINT tidak dibeli langsung, diperoleh melalui aktivitas dalam game.
— Raid: Buru bos Woodunkeu untuk "Mint Power"
— Character Sharing: Dapatkan $MINT dengan meminjamkan
Total Reward Pool: $70,000 — $10K Launch Airdrop + $60K Weekly
